Who Am I?

I'm a Master's student in Autonomy Technologies at Friedrich-Alexander-Universität (FAU), with a profound focus on exactly the trinity of modern intelligent machines: Robotics, Computer Vision, and Embedded Systems.

My work bridges cutting-edge perception with real-world physical actuation. On the software side, I build robust deep learning inference pipelines, implement YOLO-based real-time detection, and orchestrate complex kinematics using ROS2 and MoveIt2. I thrive on the challenge of granting intelligence to machines moving autonomously in complex environments.

On the hardware side, I dive deep into bare-metal firmware, custom PCB design, and sensor fusion using EKF and Nav2. By mastering both the high-level cognitive algorithms and the low-level motor controllers, I engineer comprehensive, "full-stack" robotic architectures from the circuit board to the cloud.

Franka Panda — Vision-Guided Pick & Place Simulation

Full ROS2 + MoveIt2 simulation of the Franka Emika Panda 7-DOF robot arm performing autonomous colour-targeted pick-and-place in Ignition Gazebo (Fortress). A custom ColorDetector ROS2 node processes live camera images using HSV segmentation and TF2 transforms to localise Red, Green, and Blue objects in the robot's base frame. A GraspManager node monitors gripper joint states and dynamically creates/removes fixed joints in Gazebo to reliably attach and release objects — solving the physics engine grasping limitation. The full pipeline is orchestrated via a single launch file: Gazebo → ros2_control → MoveIt2 → Vision → Grasp → Pick & Place.

HSV Colour Detection

Dual-range red HSV detection + TF2 camera-to-base-frame transform for precise 3D object localisation

MoveIt2 Motion Planning

Cartesian trajectory planning with configurable hover & approach Z offsets; 10s startup delay for system readiness

Grasp Manager

Custom joint attach/detach node via Ignition service CLI — enables reliable object holding during arm motion
